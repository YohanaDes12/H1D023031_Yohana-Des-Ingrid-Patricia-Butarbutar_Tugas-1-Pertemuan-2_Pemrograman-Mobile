# Tugas-1-Pertemuan-2_Pemrograman-Mobile

## 📝 Deskripsi Singkat
Aplikasi ini dibuat sebagai **tugas pertama** pada pertemuan ke-2 mata kuliah **Praktikum Pemrograman Mobile**.
